Viral Genome MSA

Project summary

1. NCBI Entrez 모듈에 접속해 바이러스의 게놈 데이터 입수

ChEMBL EDA-Kinase inhibitor

Project summary

ChEMBL에서 Kinase inhibitor와 관련 있는 약물들의 데이터를 받아서 EDA를 진행했음

Kinase: Protein kinase. 어떤 단백질이나 화합물에 인산기를 부착하는 효소입니다. 신호 전달에 굉장히 중요하고, 다양한 곳에서 다양한 기질에 인산기를 붙이는 게 이들의 일입니다.

protein kinase에 문제가 생기면 질병이 되는 경우도 있습니다. 암도 그 중 하나지요.

Data information

Resource: ChEMBL

Preprocessing

1. Max Phase 범주화

Max Phase는 해당 약물의 임상 단계를 의미함. 4: 승인됨, -1: 실패함, 0~3: 임상 진행중

2. 분자 종류에 따라 범주화

3. 분자량이 500보다 큰가, 작은가 여부로 범주화

리핀스키의 rule of 5 마지노선이 500입니다.

Associated statics

1. ANOVA

2. Tukey HSD

ANOVA의 후속 분석

MID project

Project summary

Kaggle의 포르투갈 은행 텔레마케팅 데이터셋을 통해 주요 고객층(20~40대)의 가입 전환률을 분석하고, 가입 전환률을 높일 방안을 도출해보자.

Why? 이들이 은행측의 주요 타겟인 반면 가입 전환율이 제일 낮았음을 EDA를 통해 확인하였음.

Role

1. Data preprocessing

2. static analysis

Process

Data preprocessing

1. 이상치 확인 (별도 처리 없음)

2. 특정 조건에 따른 범주화 컬럼 생성

대출유무, 연령대 등

Static analysis

1. Chi-square

2. 맨-휘트니 U 검정

3. 2-way ANOVA+Tukey HSD

ANOVA에서 귀무가설을 기각하여 후속분석인 Tukey HSD를 진행함

4. FAMD

데이터가 수치형, 범주형 혼합형이라 FAMD를 사용함. FAMD는 차원 두 개로 진행하였음.
