Viral Genome MSA

Project summary

NCBI에서 특정 바이러스(한타바이러스, 인플루엔자 H3N2)의 게놈 데이터를 가져온 다음 MSA를 분석하고 Phylogenic tree를 생성

Process

1. NCBI Entrez 모듈에 접속해 바이러스의 게놈 데이터 입수

2. MUSCLE MSA를 진행

3. MSA 데이터를 바탕으로 Shanon entropy를 도출하여 변이 핫스팟 탐색

4. Phylogenic tree 생성

Associated statics

1. 맨-휘트니 U 검정

2. 군집분석(k-medoid)

3. (인플루엔자) 스피어만 상관분석

인플루엔자는 H3N2로 아종이 고정되어있어서 맨-휘트니 U 검정이 불가능한 관계로 스피어만 상관분석을 진행했음

Lung cancer Lifelines

Project summary

소세포성, 비소세포성 폐암의 생존 기간과 폐암의 암종별 생존 곡선을 비교

Process

ChEMBL EDA-Kinase inhibitor

Project summary

ChEMBL에서 Kinase inhibitor와 관련 있는 약물들의 데이터를 받아서 EDA를 진행했음

Kinase: Protein kinase. 어떤 단백질이나 화합물에 인산기를 부착하는 효소입니다. 신호 전달에 굉장히 중요하고, 다양한 곳에서 다양한 기질에 인산기를 붙이는 게 이들의 일입니다.

protein kinase에 문제가 생기면 질병이 되는 경우도 있습니다. 암도 그 중 하나지요.

Data information

Resource: ChEMBL

Preprocessing

1. Max Phase 범주화

Max Phase는 해당 약물의 임상 단계를 의미함. 4: 승인됨, -1: 실패함, 0~3: 임상 진행중

2. 분자 종류에 따라 범주화

3. 분자량이 500보다 큰가, 작은가 여부로 범주화

리핀스키의 rule of 5 마지노선이 500입니다.

Associated statics

1. ANOVA

2. Tukey HSD

ANOVA의 후속 분석
